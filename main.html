Soda Socrata Web Data Connector for Tableau

This web data connector provides a connection between the Soda Socrata database and Tableau server. the web data connector will allow you to access, view and analyze data from a secure endpoint in the Soda Socrata Network.

To connector your Soda Socrata Data to Tableau, Please authorize user first by entering login credientials, the soda socrata website will authenticate the previously authorized user and issue an access token. submit your API ENDPOINT URL and FORMAT to the soda socrata database server.
